Instructions:

General Instructions

  1. Identify and locate the files you need to turn in for this assignment. If in doubt, read the assignment's "Handing In" section for an explanation.
  2. Click the Browse button and select all the files you need to turn in. Hold down the SHIFT key while clicking on files to select multiple files.
  3. Verify you have included all information (including README.txt if necessary), agree to the Collaboration Statement, and then press submit. Make sure the receipt on the next page lists all the files you needed to turn in.

File Formats

  • Written assignments must be turned in as text files or PDFs; do NOT hand in Word / Pages documents.
  • Programming assignments should include any files necessary to run your program as well as a text file named README.txt (see below).

README.txt

Every programming assignment requires a README.txt; written assignments do not unless you collaborated with someone. A README.txt should include:

  • A list of the students/groups with whom you collaborated in any way on this assignment (Be honest! Failure to disclose this information is a violation of the Collaboration Policy.)
  • Any extra information mentioned in the assignment (check the "Handing In" section for a list).
  • (If code:) A description of any possible bugs or problems with your program (rest assured, you will lose fewer points for documented bugs than undocumented ones).
  • (If code:) Any special instructions required to run your project.
  • (If code:) A description of any extra features you chose to implement.

Upload Limits:

  • No more than 10MB total for any assignment.
  • No more than 100 individual files per assignment.
